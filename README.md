# java-jbmp

### Descripción
Simple editor de imagen en formato bmp.

### Caracteriticas
  1- Permite realizar las siguientes operaciones sobre una imagen BMP.
  * Open.
  * Save.
  * Reset.
  * Grayscale/Negative/Sepia/Blur/Edges.
  * Vertical/Horizontal/Left/Right.

### Tecnologías
  * JavaSE8
  * Swing
